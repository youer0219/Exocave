# ExoCave

离开洞穴

[第2届godot游戏创作节参赛作品](https://itch.io/jam/httpsgithubcomli-game-academy-craft-2)

## 核心机制

以重力翻转为核心的解密游戏，虽然中间一些步骤需要很好的操作。

## 游玩

网页端游玩与Windows打包下载：[itch链接](https://youer0219.itch.io/exocave)

## 引擎版本

Godot4.4.1稳定版
